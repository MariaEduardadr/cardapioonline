# 🍔 BurgerMenuOnline

> cardápio online para uma hamburgueria

<p>BurgerMenuOnline é um site que permite aos clientes adicionar os produtos no carrinho, analisar o pedido, e enviar o pedido diretamente 
para o WhatsApp de forma rápida e fácil.</p>

## 🛠 Tecnologias Utilizadas
- HTML
- CSS
- JavaScript
- Tailwind CSS
